# Golang Boilerplate Using DDD Structure


## clone the project
```bash
git clone https://github.com/pratham704/golang-ddd && cd golang-ddd
```

## install dependencies
```bash
make install
```

## run the project [development mode]
```bash
make dev
```

## build the project
```bash
make build
```

## run the project [production mode]
```bash
make run
```
